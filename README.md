## What is
AndroidアプリにおけるClean Architectureの実装サンプルです。  
Qiitaの投稿情報を全件取得・タグ別に取得し、表示します。

## WebAPI
[Qiita API v2](https://qiita.com/api/v2/docs)を使用しています。  
APIの仕様変更により、プログラムが動作しなくなる可能性があります。

- Retrofit2
